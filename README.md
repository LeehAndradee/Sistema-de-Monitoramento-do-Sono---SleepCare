# SleepCare – Sistema de Monitoramento de Sono

O SleepCare é um sistema desenvolvido para a disciplina de Engenharia de Software, com o objetivo de auxiliar o usuário no monitoramento e melhoria da qualidade do sono.

O sistema permite que o usuário registre diariamente seus horários de sono (hora que dormiu e hora que acordou), realizando automaticamente o cálculo do total de horas dormidas. Além disso, gera relatórios semanais e disponibiliza uma área com informações e dicas para promover hábitos de sono mais saudáveis.

O projeto foi desenvolvido utilizando Python, Django e PostgreSQL, aplicando conceitos de análise, modelagem e desenvolvimento de sistemas.
